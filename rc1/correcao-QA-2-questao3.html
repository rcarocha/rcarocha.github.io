Um protocolo confiável na camada de transporte precisa enviar 100 pacotes para outra estação, com erro de transmissão de 50% dos pacotes transmitidos (ignore erros nos ACKs). . Considere os seguintes parâmetros:

  • 100 pacotes para serem entregues
  • R = tempo de ida e volta
  • T = timeout
  • t = tempo de transmissão de cada pacote na origem (colocação dos bits na rede)

Quanto tempo demora para transmitir esses pacotes? Considere três possíveis abordagens de correção de erros: Bit Alternado e Go-Back-N com retransmissão integral e seletiva. Considere que a janela de transmissão é maior que os 100 pacotes. Não também nenhum impeditivo na transmissão, como largura de banda de transmissão ou congestionamento, e há simetria nos caminhos dos pacotes.

Bit alternado

O tempo de transmissão para os pacotes transmitidos com sucesso é 50(t+R). O tempo de transmissão para os pacotes retransmitidos é 50(t+R+T), totalizando 50(2t+2R+T).

Go-Back-N, retransmissão seletiva

Se a janela de transmissão é maior que os 100 pacotes, então eles serão totalmente transmitidos sem que tenha sido necessário o recebimento de nenhum ACK.

No Go-Back-N com retransmissão seletiva, o tempo para transmitir é:

    100t (pacotes transmitidos com sucesso)
    + 50t (pacotes retransmitidos)
    + T (após timeout é que retransmissão é iniciada) 
    = 150t + T

Go-Back-N, retransmissão integral

  • Melhor caso - últimos 50 pacotes perdidos (e não mais perdidos), que é igual da retransmissão seletiva
  • Pior caso - apenas primeiro pacote é perdido a cada retransmissão, fazendo com que todos os pacotes seguintes devam ser retransmitidos, ocasionando a seguinte situação:
    1. 1ª transmissão: P1 é perdido e todos os demais descartados do receptor
    2. após timeout do P1, pacote P2 é perdido na retransmissão, e todos os seguintes são descartados no receptor
    3. após timeout do P2, pacote P3 é perdido na retransmissão, e todos os seguintes são descartados no receptor
    4. após timeout do P3, pacote P4 é perdido na retransmissão, e todos os seguintes são descartados no receptor
    5. e assim por diante, até perder o pacote P50 (para garantir a perda de 50% dos pacotes. Os pacotes de P51 a P100 são transmitidos depois com sucesso.

A cada passo temos o tempo T+t (o tempo de transmissão dos demais pacotes não influencia, mas apenas o timeout). Como isso ocorre com 50 pacotes, temos 50(t+T). Para a transmissão dos demais precisamos de 50t, totalizando

    50(2t + T)

No pior caso.

Resposta

  • Bit alternado: 50(2t+2R+T).
  • Go-Back-N retransmissão integral: 50(2t + T) <= tempo <= 150t + T
  • Go-Back-N retransmissão seletiva: 150t + T
