Questão 1

Quando uma estação envia uma requisição a um servidor de DHCP com endereço IP IP_DHCP, qual é o endereço IP de origem e destino do pacote IP que sai da estação?

Opções

  1. 0.0.0.0, 255.255.255.255
  2. IPestação, IPDHCP
  3. 0.0.0.0, IP_DHCP
  4. 255.255.255.255, IP_DHCP
  5. IP_estação, 255.255.255.255
  6. 255.255.255.255, 255.255.255.255

Questão 2

Uma estação está ligada em uma rede local com mais 3 estações e um roteador, que possui uma interface de rede adicional ligada ao restante da Internet. Qual é o número máximo de resoluções ARP (ARPRequest) que esta estação poderá fazer?

Opções

  1. 4
  2. 1
  3. 3
  4. 5
  5. 6
  6. não há limite: será feita uma resolução por endereço IP com o qual a estação conversar (em teoria, infinitas)

Considere A enviando um pacote IP para B.

Qual é o endereço MAC de origem do pacote de enlace que sai do roteador R rumo a B?

Opções

  1. 03-00-00-00-00-ab
  2. 01-00-00-00-00-ab
  3. 02-00-00-00-00-ab
  4. 05-00-00-00-00-ab
  5. 06-00-00-00-00-ab
