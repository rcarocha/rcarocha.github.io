Questão 1

Qual é a motivação para utilização de máscaras de rede?

Opções

  1. organizar livremente os conjuntos de endereços recebidos em mais de uma rede
  2. aumentar o número de endereços em uma rede
  3. permitir o uso de uma larga quantidade de endereços privados
  4. permitir o uso de roteadores dentro das organizações

Questão 4

Como os bits iguais a 1 de uma máscara devem ser interpretados?

Opções

  1. as mesmas posições dos bits serão utilizadas nos endereços para descrever redes/subredes.
  2. as mesmas posições dos bits serão utilizadas nos endereços para descrever estações nas redes/subredes.
  3. as mesmas posições dos bits serão utilizadas nos endereços para descrever endereços de broadcast.
  4. as mesmas posições dos bits serão utilizadas nos endereços indicam de qual classe o endereço fará parte.
