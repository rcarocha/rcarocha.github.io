  2. O Bit Alternado só envia um pacote até que ocorra o timeout, enquanto que o Go-Back-N envia até N pacotes.
  3. O Bit Alternado envia até dois pacotes até o recebimento do ACK do anterior, enquanto que o Go-Back-N envia até N pacotes.
  4. O Bit Alternado envia até dois pacotes até que ocorra o timeout, enquanto que o Go-Back-N envia até N pacotes.
  5. O Bit Alternado adota um timeout de T, enquanto que o Go-Back-N adota um timeout de N pacotes.
  6. O Bit Alternado adota um timeout de T, enquanto que o Go-Back-N adota um timeout de N/T pacotes.
  7. O Bit Alternado adota um timeout de T, enquanto que o Go-Back-N adota um timeout de T/N pacotes.
