</p> <h1>Tarefa 7 de Programação Funcional e Lógica</h1> <h1>Visão geral</h1> <p>Nesta tarefa você deverá implementar alguns predicados de manipulação de listas em Prolog.</p> <h1>Descrição</h1> <ol> <li><p>Crie um predicado <code>palindromo/1</code> que seja verdadeiro (<code>True</code>) quando uma lista passada como parâmetro seja palíndroma. Por exemplo, </p> <ul> <li><code>?- palindromo([m,a,d,a,m,i,m,a,d,a,m])</code> deve retornar <code>True</code></li> <li><code>?- palindromo([1,2,4,8,16,8,4,2,1])</code> deve retornar <code>True</code></li> </ul></li> <li><p>Crie um predicado <code>comprimeL/2</code> que retorne verdadeiro para uma lista <code>L1</code> e uma lista <code>L2</code> que seja igual à lista <code>L1</code> sem os elementos duplicados. </p> <ul> <li><code>comprimeL([a,a,a,b,b,b,c], [a,b,c])</code> deve retornar <code>True</code></li> <li><code>comprimeL([a,a,a,b,b,b,c,a],L2)</code> deve retornar <code>L2 = [a,b,c,a]</code></li> </ul></li> <li>Crie um predicado <code>duplica</code> que retorne verdadeiro para a lista <code>L1</code> e <code>L2</code>, onde <code>L2</code> contém todos os elementos da lista <code>L1</code> duplicados e na mesma ordem. <ul> <li><code>duplica([3, 5, 6],[3,3,5,5,6,6])</code> deve retornar <code>True</code></li> <li><code>duplica([3],L2)</code> deve retornar <code>L2 = [3,3]</code></li> <li><code>duplica([1,2,3],[1,1,2,E4,3,3])</code> deve retornar <code>E4 = 2</code></li> </ul></li> <li>Crie um predicado <code>replique/3</code> que receba uma lista <code>L1</code>, um inteiro <code>N</code> e uma lista <code>L2</code>, onde <code>L2</code> contém todos os elementos de <code>L1</code> replicados <code>N</code> vezes <ul> <li><code>replique([3,5,6],3,L2)</code> deve retornar <code>L2 = [3,3,3,5,5,5,6,6,6]</code></li> <li><code>replique(L1,3,[1,1,1])</code> deve retornar <code>L1 = [1]</code></li> <li><code>replique([4,5],N,[4,4,4,4,5,5,5,5])</code> deve retornar <code>N = 4</code> </li> </ul></li> <li>Crie um predicado <code>removeN/3</code> que receba uma lista <code>L1</code>, uma posição <code>N</code> e uma lista <code>L2</code> e retorne verdadeiro se a a lista <code>L2</code> é igual à lista <code>L1</code> sem o elemento que estava na posição <code>N</code> <ul> <li><code>removeN([3,5,6],3,L2)</code> deve retornar <code>L2 = [3,5]</code></li> <li><code>removeN([3,5,6],N,[5,6])</code> deve retornar <code>N = 1</code></li> <li><code>removeN([1,1,2,1,1],4,[1,1,2,1])</code> deve retornar <code>True</code></li> </ul></li> <li>Crie um predicado <code>divida/3</code> que receba uma lista <code>L1</code>, uma posição <code>N</code> e uma lista <code>L2</code> e retorne verdadeiro se a lista <code>L2</code> tiver uma lista <code>L21</code> na primeira posição e uma lista <code>L22</code> na segunda posição tal que <code>L21</code> tem os elemento de 1 a <code>N</code> da lista <code>L1</code> e a lista <code>L22</code> os demais elementos da lista <code>L1</code>. <ul> <li><code>divida([1,2,3,4],2,[[1,2],[3,4])</code> deve retornar <code>True</code></li> <li><code>divida([1,2,3,4],3,[L21,L22])</code> deve retornar <code>L21 = [1,2,3] , L22 = [4]</code></li> <li><code>divida(L1,2,[[1,2],[3,4])</code> deve retornar <code>L1 = [1,2,3,4]</code></li> <li><code>divida(L1,3,[[1,2],[3,4])</code> deve retornar <code>False</code></li> </ul></li> <li>Crie um predicado <code>fatia/4</code> que receba uma lista <code>L1</code>, uma posicao (inteiro) <code>P1</code>, uma posição <code>P2</code> e uma lista <code>L2</code>, e retorne verdadeiro caso a lista <code>L2</code> seja formada dos elementos de <code>L2</code> da posição <code>P1</code> até a posição <code>P2</code> (ou seja, deve ser uma fatia de <code>L1</code> de <code>P1</code> a <code>P2</code>) <ul> <li><code>fatia([1, 2, 3, 4, 5, 6, 7],3,6,[3,4,5,6])</code> deve retornar <code>True</code></li> <li><code>fatia([1, 2, 3, 4, 5, 6, 7],1,3,L2)</code> deve retornar <code>L2 = [1,2,3]</code></li> <li><code>fatia([1, 2, 3, 4, 5, 6, 7],P1,7,[6,7])</code> deve retornar <code>P1 = 6</code></li> </ul></li> </ol> <h1>Requisitos</h1> <p>Reveja o <a
